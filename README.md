# Telco Churn Analysis

This project explores churn for a telecommunications company, Telco. The goal is to identify high-risk customer groups, explore churn drivers, and provide potential business solutions.


## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- NumPy
- Jupyter Notebooks
